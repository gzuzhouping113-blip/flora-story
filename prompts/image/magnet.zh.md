# 整体画面要求
生成一张 3:4 比例的竖版高清图像。背景为干净的纯白、浅灰或极浅暖白色，不出现桌面、房间、包装纸等环境杂物。

# 核心主体与还原控制（最高优先级）
必须严格基于用户上传的原图进行艺术化转换。
1. 构图忠实：保持原图中主要花朵的种类、数量、大小比例、相对位置和遮挡关系。
2. 不增不减：不要添加原图中没有的花朵、叶片、蝴蝶、文字、丝带或装饰。
3. 主体完整：原图中的核心花朵主体必须完整保留，不要大幅裁切。

# 风格化转换细节（艺术冰箱贴 / 手帐贴纸）
在忠实原图构图的基础上，把花束转换成一枚高级感的艺术贴纸。
1. 美术风格：偏手帐贴纸、植物插画、轻量装饰画，而不是逼真的树脂产品照。整体要柔和、精致、有设计感和收藏感。
2. 插画质感：花瓣和叶片以手绘色块、柔和渐变和细腻线条表现，可以有轻微浮雕层次，但整体必须明显是艺术插画，不要照片级真实。
3. 反光控制：不要强烈塑料反光、陶瓷质感、玻璃质感、金属质感、真实水珠和摄影棚高光。若原图有水滴，只用少量手绘亮点概括，不要画成逼真水珠。
4. 轮廓边框：主体外沿有一圈自然、圆润、干净的暖白色描边，描边贴合整体轮廓，不要过粗，不要像厚重实物边框。
5. 色彩氛围：保留原花束主要色彩倾向，同时降低饱和度，使用柔和、通透、温柔的色彩；画面应明亮、清爽、有呼吸感。
6. 光影表现：只有非常轻柔的平面投影和浅浅悬浮感，避免强烈立体建模和过度 3D。
