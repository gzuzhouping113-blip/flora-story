# 整体画面要求
生成一张 3:4 比例的竖版治愈系花卉插画。背景为纯白或略偏暖的浅色水彩纸纹理，画面干净、留白充分，不出现环境背景。

# 核心主体与还原控制（最高优先级）
必须严格基于用户上传的原图进行艺术化重绘。
1. 构图锁定：保持原图中主要花朵数量、位置、遮挡关系和整体姿态。
2. 内容忠实：保留原图花朵品种特征和核心色彩倾向，不把花种画成别的花。
3. 严禁加戏：不要添加原图中不存在的飞鸟、蝴蝶、额外花草、文字或装饰。

# 人物与肢体排除（最高优先级）
画面只能出现花束、花朵、叶片和必要的植物结构。严禁出现任何人物相关元素：人脸、手、手指、手臂、身体、背影、剪影、皮肤、衣物、人物照片、人物倒影或人物阴影。即使原图里有手持花束、人物背景或身体局部，也必须将这些人物元素完全移除或弱化为干净背景，只保留花束主体。

# 风格化转换细节（清透水彩花卉）
在锁定原图构图的前提下，将画面转换为更有美感的水彩花卉插画。
1. 笔触质感：使用轻盈、透明、层次丰富的水彩笔触，保留自然晕染、水痕和纸张纹理。
2. 细节处理：简化照片中过于琐碎或凌乱的细节，用优雅的色块和线条概括花瓣层次，不要画成照片滤镜。
3. 色彩氛围：整体温柔、明亮、清透，色彩有艺术性的灰度和层次，避免过艳、过脏或过暗。
4. 构图留白：主体周围保留适度留白，像一本精致植物图鉴或手帐中的花卉插画。
5. 情绪表达：画面要有安静、浪漫、被珍藏的感觉，不要商业海报感。
