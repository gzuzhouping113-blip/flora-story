# 整体画面要求
生成一张 3:4 比例的竖版高清图像。画面只出现一张居中的拍立得纪念相片，外部背景必须是纯白、浅灰或浅米色的干净平面。

# 核心主体与拍立得相框生成
画面中央是一张完整的复古拍立得纸质相片。相框为温润的白色纸张质感，下方保留经典宽留白区域，但不要添加任何文字。拍立得之外不允许出现桌面、布料、墙面、夹子、胶带、贴纸、手、花瓣、光斑或其他装饰物。

# 人物与肢体排除（最高优先级）
画面只能出现拍立得相片、花束、花朵、叶片和必要的植物结构。严禁出现任何人物相关元素：人脸、手、手指、手臂、身体、背影、剪影、皮肤、衣物、人物照片、人物倒影或人物阴影。即使原图里有手持花束、人物背景或身体局部，也必须将这些人物元素完全移除或弱化为干净背景，只保留花束主体。

# 画面内容还原与艺术滤镜控制
拍立得相框内部的照片区域必须基于用户上传原图，保留花束主体和构图关系。
1. 构图忠实：保留原图中花朵的主要内容、数量、位置和整体姿态，不随意增减元素。
2. 背景简化：照片区域内若原图背景杂乱，请在不改变花束主体的前提下弱化背景，让花束更突出。
3. 艺术胶片感：不是粗糙旧照片，也不是重度发黄滤镜；使用柔和电影胶片色调、轻微颗粒、温暖高光和非常克制的暗角。
4. 美感优化：让花束看起来像被认真收藏的一张纪念照片，色彩更柔和，层次更优雅，画面更有情绪。
5. 纸张质感：拍立得边框只保留轻微纸张纹理和浅阴影，不要过度 3D，不要像真实商品摄影。
6. 极简克制：整张图保持安静、留白、明亮，不要添加手写字、日期、水印、贴纸、夹子、胶带或额外装饰。
